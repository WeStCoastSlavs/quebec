Top-level domain

From the Simple English Wikipedia, the free encyclopedia that anyone can change

A top-level domain is an extension at the very end of a domain name, which is a name for an Internet website.

Examples:

  • .com (Commercial)
  • .net (Network)
  • .org (Organization)
  • .uk (United Kingdom)
  • .gov (Government)
  • .edu (Education)
