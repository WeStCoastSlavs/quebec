User:Byrialbot

From the Simple English Wikipedia, the free encyclopedia that anyone can change

This user is a bot run by Byrial (talk).

It is used for helping to make many small changes that would take a long time for a person to do alone.
Administrators: if this bot isn't working right or doing bad things, please block it.

This is a robot account which is used by User:Byrial.

  • Owner: da:User:Byrial
  • Contact: da:User talk:Byrial
  • Activities: Interwiki working from the Danish (da:), Swedish (sv:) and Norwegian (no: and nn:) Wikipedias. I often find the pages to be changed from offline analysis of database dumps, and tries to solve conflicts which are left by the automatic running interwiki bots..
  • Operation mode: Mostly manually assisted. May run automatically in some cases.
  • Program: Pywikipediabot
  • Created: June 11, 2007
  • Status: Bot flag at June 12, 2007
  • Wikipedias with bot flag at June 11, 2007: ar:, da:, en:, eo:, fo:, he:, is:, it:, nds:, nn:, no:, sv:, uk:, vo:
  • Logs: about, for, contributions
