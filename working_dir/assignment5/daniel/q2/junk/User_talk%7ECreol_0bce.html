[change] Template:Cookie

I have a question for you about this template. Instead of having it say Somebody has awarded you a cookie, could you change it to say the name of the person who gave the other person the cookie?

Example: Razorflame has given you a cookie. Now enjoy it! (If I had given you a cookie).

Is such a thing possible? If it is, then I am at a complete loss because I am not a template maker. Cheers, Razorflame 13:42, 28 April 2008 (UTC)

It gets a little tricky as there are two possible ways to do it and there may be a third but the third one is the one you are actually looking for. Searching through the variables and magic words, I can't find a variable used for the sender name function. It may exists, it may not. It would be the best option though. The other two options which do exist:
  1. Using the signature ( ~~~ = the -- Creol(talk) part of the signature). This works for many accounts (not mine) but can get a little screwy at times depending on how the signature is designed. This option would be better for a personal user space based template where all the users that use it know they have to have a specific format to their signature.
  2. Using an optional variable. {{Cookie |giver=some user}} would work. Using the default setting {{{giver|Somebody}}} would allow for it to work with or without the giver option (or name or from or whatever).
-- Creol(talk) 23:05, 28 April 2008 (UTC)
Do you think that the second option would work? That's the option that I think would work the best of the available options (because you said the 3rd didn't work...), but what do you think? Cheers, Razorflame 14:51, 29 April 2008 (UTC)
Sorry it took so long to get back to this one. The second option not only is workable, but it was a part of the template from the first version you made. With or without the name option, the first option works fine ( {{cookie|name=Razorflame}} or {{Cookie|Razorflame}} both work). There was/is an issue with linking the name under certain usages. This may or may not have been corrected (it linked Somebody or when not using name but didn't link name entries)

. I can not recall if this has been corrected and actually it should link on all but "Somebody" rather than the other way.. -- Creol(talk) 05:54, 10 May 2008 (UTC)
